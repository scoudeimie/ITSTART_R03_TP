Authentification
	-> La base de données doit être disponible
	-> Se connecter à la base de données en PHP
	-> Créer la requête SQL pour récupérer le login/mot de passe
	-> Comparer les informations avec celles saisies dans le formulaire
	-> Renvoyer la page de l'application si authentification OK
	-> Renvoyer au formulaire d'authentification sinon

Créer la page de l'application
	-> Créer la page HTML
		-> ne pas mettre le champs de saisie de message dans le cas d'un visiteur
		-> liste des salons
		-> liste des utilisateurs connectés au salon sélectionné
		-> liste des messages du salon sélectionné
	-> Créer le fichier CSS
	-> Créer le script Javascript d'interactivité
	-> La page ne doit être affichée que si l'utilisateur a un profil
